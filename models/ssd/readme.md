| name | for What | reference |
| ---  | ---      |    ---    |

|  jaccard overlay  |  to get best box   |  a.k.a. [IOU](https://cdn-images-1.medium.com/max/800/1*_Xf5FUbuUgq8GNyITM3Dwg.png)  |
